# 🏥 Healthcare Data Analysis & Machine Learning Project

## 📌 Project Overview
This project analyzes hospital utilisation, capacity efficiency, and patient flow using encounter-level healthcare data. It applies statistical analysis and machine learning techniques to identify operational inefficiencies and improve healthcare decision-making.

## 🎯 Objectives
- Analyze hospital admissions, ED visits, and bed utilization trends
- Identify capacity bottlenecks using Length of Stay (LOS)
- Detect seasonal variations in healthcare demand
- Study patient flow and readmission patterns

## 📊 Dataset
Synthea synthetic healthcare dataset (encounter-level data)

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn (ML models)
- Jupyter Notebook

## 🤖 Machine Learning Methods
- Logistic Regression (Supervised Learning)
- K-Means Clustering (Unsupervised Learning)
- PCA (Dimensionality Reduction)

## 📈 Key Insights
- Inpatient encounters consume the highest bed-days
- LOS variability is a major driver of capacity pressure
- Seasonal variation exists in specific diagnosis groups
- 7–30 day readmissions are most frequent
